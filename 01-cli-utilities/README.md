# 01 — CLI Utilities on Go

Набор консольных утилит на Go, демонстрирующих работу со стандартной библиотекой:
чтение и валидация пользовательского ввода, обработка строк, работа с коллекциями
(`map`, `slice`), сортировки, пользовательские типы ошибок и форматирование вывода.

Каждая задача — самостоятельная программа в пакете `main`, читающая данные из
стандартного ввода (`stdin`) и печатающая результат в стандартный вывод (`stdout`).

## Цель

Освоить базовый инструментарий Go: модель ввода-вывода, парсинг и валидацию данных,
идиоматическую обработку ошибок (включая собственные типы ошибок и `errors.As`),
а также покрытие чистой логики юнит-тестами.

## Состав

| Каталог  | Задача | Что реализовано |
|----------|--------|-----------------|
| `task1`  | Калькулятор | Чтение двух операндов и операции (`+ - * /`), вычисление результата, повторный запрос делителя при делении на ноль, корректная печать целых и дробных значений. |
| `task2`  | Top-K частотных слов | Подсчёт частоты слов, выбор `K` самых частых; при равной частоте — лексикографический порядок. Логика вынесена в функцию `TopKFrequent` и покрыта тестами. |
| `task3`  | Пересечение множеств | Парсинг двух списков чисел и вывод их пересечения без дубликатов, с сохранением порядка появления в первом списке. |
| `task4`  | Журнал визитов | In-memory хранилище визитов пациентов с командами `Save`, `GetHistory`, `GetLastVisit`. Использует пользовательский тип ошибки `PatientNotFoundError` и парсинг дат `time.Parse`. |

## Запуск

Требуется Go 1.25+.

```bash
# Калькулятор
go run ./task1

# Top-K частотных слов (первая строка — слова, вторая — число K)
go run ./task2

# Пересечение списков чисел
go run ./task3

# Журнал визитов (команды читаются построчно из stdin)
go run ./task4
```

### Пример: task4 (журнал визитов)

```
Save
Ivanov
cardiologist
2023-01-10
GetLastVisit
Ivanov
cardiologist
```
Вывод:
```
2023-01-10
```
