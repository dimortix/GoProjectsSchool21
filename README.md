# Go Projects

Сборник учебных проектов на Go — от базовых консольных утилит до конкурентного
программирования и backend-сервиса с чистой архитектурой. Каждый проект
самодостаточен, имеет собственный `go.mod` и подробный README.

## Проекты

| № | Проект | Описание |
|---|--------|----------|
| [01](./01-cli-utilities) | **CLI Utilities** | Консольные утилиты на стандартной библиотеке: калькулятор, частотный анализ слов, пересечение множеств, журнал визитов с пользовательскими ошибками. |
| [02](./02-concurrency) | **Concurrency** | Конкурентность в Go: синхронизация горутин, пайплайны на каналах, тикер с обработкой сигналов, потокобезопасный дженериковый LRU-кэш. |
| [03](./03-tictactoe-api) | **Tic-Tac-Toe HTTP API** | HTTP-сервис крестиков-ноликов с непобедимым ИИ (minimax), чистая архитектура и DI на Uber FX. |

## Стек

- **Go 1.25**
- Стандартная библиотека (`net/http`, `sync`, `container/list`, `time`, дженерики)
- [go.uber.org/fx](https://github.com/uber-go/fx) (проект 03)

## Быстрый старт

```bash
git clone https://github.com/dimortix/GoProjectsSchool21.git
cd GoProjectsSchool21

# Каждый проект запускается из своего каталога, например:
cd 01-cli-utilities && go run ./task1
```

Подробности по запуску и тестированию — в README соответствующего проекта.
